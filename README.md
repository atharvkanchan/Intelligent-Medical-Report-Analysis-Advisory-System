# Intelligent-Medical-Report-Analysis-Advisory-System
# 🏥 AI Medical Report Analyzer

## 📌 Overview
An AI-powered system that analyzes medical reports using NLP techniques.

## 🚀 Features
- PDF & Image Processing (OCR)
- AI Summarization
- Risk Detection
- Medical Q&A Chatbot
- Interactive UI

## 🧠 Tech Stack
- Hugging Face Transformers
- FastAPI (Backend)
- Streamlit (Frontend)
- Python

## ▶️ Run Project

### Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend
cd frontend
pip install streamlit requests
streamlit run app.py

## ⚠️ Disclaimer
This system is assistive and not a replacement for medical professionals.
